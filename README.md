# ERP-Based-Sales-Distribution-Management-System
ERP-based Sales and Distribution Management System built using Java and MySQL. Manages customers, products, orders, inventory and billing with role-based access and automated stock updates to improve accuracy and reduce manual effort.
ERP-Based Sales & Distribution Management System

This project is designed to simplify and automate the core processes involved in sales and distribution for small and medium-scale businesses. The system manages customer records, product inventory, order creation, billing, and sales tracking, ensuring accuracy and efficient workflow management.

Key Features:

Customer data and account management

Secure authentication and data validation

MySQL database with normalized tables for customers, products, orders, and billing details.
